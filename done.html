Concept

A productivity app to manage daily tasks on the go.

  • Role: UI Designer
  • Period: March 2020
  • Tools: Sketch, InVision

Context

This project is the first project completed as a UI design student.

Brief

Produce at least four screens of a to-do list mobile app from low to high fidelity UI design. Identify key functionalities and create a mood board to integrate UI elements and typography.

Overview

Done is a simple, easy-to-use mobile application to manage daily tasks and get things done.

Objectives

  • Keep users on track by allowing access to all existing tasks at a glance whether for the day or as a whole.
  • For any day give users a clear vision on what is done and what remains to be done.
  • Allow users to organized tasks at will.
  • Users should be able to add, delete or modify tasks.
